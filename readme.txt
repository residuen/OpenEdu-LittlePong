OpenEdu-LittlePong ist ein OpenSource Projekt
Es findet Einsatz im Informatik-Unterricht

Mögliche Erweiterungen als Unterrichts/Projektinhalt:
    2,- 3,- oder 4 Schläger integrieren
    Ballspeed steigt mit der Spieldauer
    Zeit- und Punkteanzeige
    Variabler Abprallwinkel des Balls von Schläger
    Objekte "abschießen" für Extras, z.B. variable Ballzahl, veränderbare Schlägergröße, etc.
    Einstellungsmenü für: Ballgröße, Startgeschwindigkeit, Schlägergröße, Sondermodi, ... etc.

Programmiersprache:
OpenEdu-LittlePong ist mit der Programmiersprache Java entwickelt worden.
OpenEdu-LittlePong setzt Java 1.6 oder hoeher vorraus.<br>
